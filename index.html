Regulatory Finance Software

Cost of capital analysis for utility cases, built to be audited instead of hand-waved.

TURBOROE is a focused effort to automate cost of capital analysis for water and gas distribution utilities without hiding the underlying assumptions. The aim is simple: produce a workflow another analyst can rerun, inspect, and defend.

What This Is

A public-facing commitment page for a private working codebase.

The software itself is still private while the project is pushed to a minimum viable, analyst-usable state. This page exists to make the goal concrete now instead of waiting until everything is polished.

The underlying project is designed to move from raw market and filing data to transparent Excel outputs that keep formulas visible, assumptions reviewable, and the analytical path easy to challenge.

Methods

Four ways to estimate cost of equity, one auditable workflow.

CAPM

OLS beta, Blume adjustment, Hamada unlevering, and both book and market capital structure.

DCF

Constant-growth and multi-stage dividend discount models with analyst and historical inputs.

Comparable Earnings

Market-to-book versus forward ROE regression for a market-based return cross-check.

Risk Premium

Yield-conditioned utility premium estimates using bond benchmarks, proxy indices, and HAC errors.

Minimum Viable Project

The MVP is reached when another analyst can install, configure, run, and inspect the workflow without needing the original author to translate the project for them.

  • Finalize a stable main orchestration flow
  • Add startup validation for environment variables and required source files
  • Centralize tunable parameters in one documented place
  • Reduce startup friction to a short, realistic quick-start path
