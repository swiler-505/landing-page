# TURBOROE Landing Page

Standalone public landing page for the TURBOROE project.

## Purpose

This repo is meant to stay public even while the main TURBOROE codebase remains private.
It is a lightweight accountability and marketing site, not the application itself.

## Publish on GitHub Pages

1. Push this repo to GitHub.
2. Go to `Settings -> Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select the `main` branch and the root folder.
5. Save.

GitHub Pages should publish the site at:

`https://<your-github-username>.github.io/<repo-name>/`
